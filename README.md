# Smart-Ticket-Booking-System-c
Ticket booking system in C with file handling, seat validation, and persistent storage. Supports booking, search, update, and cancellation.
A menu-driven ticket management system built in C that simulates real-world booking operations with persistent data storage using binary files.
🚀 Features
🎟️ Book tickets with auto-generated unique Ticket IDs
💺 Seat allocation system with duplicate prevention
🔍 Search tickets by Ticket ID
✏️ Update passenger details
❌ Cancel tickets with safe file handling
📂 Persistent storage using binary file (tickets.dat)
✅ Input validation for age and seat number
🛠️ Tech Stack
Language: C
Concepts Used:
Structures
File Handling (fopen, fread, fwrite)
Modular Functions
Input Validation
Basic Data Management System
